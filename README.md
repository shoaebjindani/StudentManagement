# Spring Boot Base Framework

This project is a **Spring Boot starter framework** designed to provide a ready foundation for building web applications. It includes essential modules so developers can quickly extend and build their own features.

## Features

* Built-in **authentication** and **authorization**
* **User, role, and permission** management screens
* Basic **UI layout structure** for creating new pages
* Modular code structure for easy extension

## Tech Stack

* Spring Boot
* MySQL
* Maven
* Deployable on **Tomcat**
* Supports **Docker image** build

## Purpose

To offer a clean, reusable base framework that speeds up development and maintains consistency across projects.
